ContextCake gives AI tools a working view of team context without flattening company policy, team practice, and local judgment into one brittle document.

If you are evaluating the product, start with the shape of the answer first, then the mechanics behind it.

## What to expect

ContextCake is an open source engine. It resolves layered context at read time, keeps provenance attached to the answer, and surfaces disagreement instead of silently blending it away.

The commercial layer lives in [Packs](/packs): profession-specific workflow kits that can travel across tools while still fitting the same context model.
