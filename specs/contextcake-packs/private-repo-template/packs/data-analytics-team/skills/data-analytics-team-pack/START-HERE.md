# Start Here

This pack helps a small data or analytics team turn repeat stakeholder requests into clarified
specs, validated builds, and useful insight summaries.

Use it as plain files, as a Claude Code plugin, or as an OKF-compatible bundle for future
ContextCake layers.

## Reading Order

1. `overview/pack-purpose.md`
2. `workflows/stakeholder-request-to-insight.md`
3. `templates/clarified-spec-template.md`
4. `templates/validation-checklist-template.md`
5. `templates/insight-summary-template.md`

## Local Overrides

Do not edit installed pack files directly. Put team-specific policies, examples, and exceptions in
your project or team's own context layer, then keep this pack clean for updates.
